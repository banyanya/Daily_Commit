# 프로그램 작성 순서

1. 소스 파일 생성 - 프로그램 작성

2. 컴파일 - 오브젝트 파일 생성
소스파일(고급언어) → 오브젝트 파일(저급언어, 기계어)

3. 링크 - 실행 파일 생성
오브젝트 파일(.obj) → 실행 파일(.exe)

4. 로딩, 실행
