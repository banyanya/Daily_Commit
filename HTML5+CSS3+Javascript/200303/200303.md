# 프로그램 작성 순서

1. 소스 파일 생성 - 프로그램 작성

2. 컴파일 - 오브젝트 파일 생성
소스파일(고급언어) → 오브젝트 파일(저급언어, 기계어)

3. 링크 - 실행 파일 생성
오브젝트 파일(.obj) → 실행 파일(.exe)

4. 로딩, 실행

# 전처리기(Preprocessor) - \#
전처리기의 내용은, 컴파일러가 **컴파일(고급언어 → 기계어)**을 수행하기 전에 먼저 처리하게 된다.



# 함수의 형태
```
int main(void)
출력형 함수명 (입력형)
```


# 꺽쇠기호와 따옴표 기호의 차이
```
<stdio.h> : 시스템 헤더 파일
"myheader.h" : 사용자 헤더 파일
```
